# Project Brief — APEX (aps_sis)

## Core Purpose
APEX (Academic Progress and Excellence) is an Odoo 18 Community Edition module for managing academic resources, student submissions, grading, AI-assisted marking, and awards in a school environment.

## Key Requirements
- Hierarchical resource management (parent/child relationships)
- Student submission and grading workflow
- AI-assisted marking with multiple model support
- Subject/category-based organization
- Course Explorer for browsing course content in a split-pane view
- Award and certificate system

## Technology Stack
- **Backend**: Odoo 18 Python ORM
- **Frontend**: OWL 2 (Odoo's JavaScript framework)
- **Database**: PostgreSQL via Odoo ORM
- **Assets**: ES modules, QWeb templates, SCSS/CSS

## Success Criteria
- Teachers can manage resources hierarchically
- Students can view and submit work
- AI marking produces accurate feedback
- Course Explorer provides intuitive content navigation
